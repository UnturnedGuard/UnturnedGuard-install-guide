>>> Установка
Процесс первой установки

1. Выключите сервер (если запущен)

2. Скопируйте контент который находится в папках Plugins и Libraries из установленного лоадера, в схожие и указанные папки: 
Жмите на вставить с заменой файлы, если просит!
Plugins -> Rocket\Plugins
Libraries -> Rocket\Libraries

3. Откройте конфигурацию плагина
- Rocket\Plugins\UnturnedGuardGoldBootstrapper.RocketMod\UnturnedGuardGoldBootstrapper.RocketMod.configuration.xml

4. Конфигурация конфига плагина
DebugMode: Нужен только для специальных случаев, менять только в том случае, если вас попросили разработчики
LicenseKey: Лицензионный ключ

<?xml version="1.0" encoding="utf-8"?>
<UnturnedGuardBootstrapperConfiguration xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <DebugMode>false</DebugMode>
  <LicenseKey>Put your license key here</LicenseKey>
</UnturnedGuardBootstrapperConfiguration>

5. Включить сервер

6. Готово!

Если вам нужна помощь, у вас есть вопрос, или хотите отправить найденных баг, то сюда: https://discord.gg/jc4FPg6H6S
